# course_selector
我们不能给你明确的队名 但可以给你明确的代码
---------------------------------
### **功能：**
* 导入培养方案，识别出指定学期的专业必修课、专业选修课、公共基础课、通识课等

--------------------------

* 搜索各院系各课程，包含老师、时间、地点、往届评分、课程听感等信息（可以考虑导入往届同学对该课程的测评
* 发布对上过课程的测评

---------------------------

* 多方面设置个人偏好
    * 对老师给分和任务量的期望
    * 对早八/午一/晚课的厌恶程度
    * 对课间通勤难度的要求
    * 对上四休三的喜爱程度
    * 对补退选的接受程度
    * etc.
* 设置个人具体喜好（eg.我一定要冲社会博弈论）
* 基于上述两点通过神经网络计算出推荐的课程（比如专业课选哪个老师，有什么推荐的通识课），生成可视化课表

-----------------------------

* 输入已选人数和限选人数进行投点指导
